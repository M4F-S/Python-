# Module 10 — Exercise Workspace

Five exercises, each a single file:

```
exercises/
├── ex0/lambda_spells.py
├── ex1/higher_magic.py
├── ex2/scope_mysteries.py
├── ex3/functools_artifacts.py
└── ex4/decorator_mastery.py
```

Run any exercise directly:

```
python3 ex0/lambda_spells.py
```

Constraints from the subject (check the module's `lesson.md` for the
full rundown):

- `Callable` must come from `collections.abc`, not `typing`.
- No `global`. Closures with `nonlocal` are the intended tool.
- No file I/O, no external libraries.
- Each exercise only authorises a small set of helpers — respect them.

Honor-system: only peek at `../solutions/` after submitting.
